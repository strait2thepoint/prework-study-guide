HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is a container for the meta elements
  • The meta elements contains information about the webpage that is not visible on the website

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd is to move to the base directory
  • cd "x" is to move to the x directory
  • dir is to see what is in the directory
  • git add -A is to stage changes
  • git commit -m "x" adds a message to our commit
  • git pull origin main pulls base branch into your feature branch
  • git push origin and our branch name will merge our feature branch to the main
